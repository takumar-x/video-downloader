## üá∫üá∏ English
**Installation Instructions:**

1. Download the latest version of `yt-dlp.exe` from this link:  
   [https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe](https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe)

2. Move the downloaded `yt-dlp.exe` file directly into the `video-downloader` folder (where the program executable is located).

3. Download the latest version of `ffmpeg.exe` from this link:  
   [https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip](https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip)

4. Open the archive. Go inside the `bin` folder.

5. Find the `ffmpeg.exe` file.

6. Copy `ffmpeg.exe` directly into the `video-downloader` folder.
